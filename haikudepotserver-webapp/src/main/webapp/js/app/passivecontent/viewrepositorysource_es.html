El origen de repositorio define una URL desde la cual se puede obtener la información del repositorio. Generalmente hay múltiples URL para un repositorio, uno para cada arquitectura.

Adicional al contenido del repositorio, también existe un archivo repo.info con un campo url. Este es un identificador único para dicho origen de repositorio. Esta es la clave que une los datos de HDS con el repositorio en la aplicación cliente de escritorio HaikuDepot.

Propósito

Esta aplicación web es una herramienta en línea para trabajar con los paquetes de software del Sistema Operativo Haiku (Haiku). La aplicación web tiene por objetivo proveer la siguiente funcionalidad:

  • Un catálogo de paquetes accesible vía internet.
  • Un repositorio de información adicional relacionada a los paquetes que no se incluye en el formato estándar de los paquetes — como:
    • Capturas de pantalla
    • Iconografía
    • Traducciones
    • Calificaciones de usuarios
  • Una interfaz de usuario para manipular la información adicional.
  • Un mecanismo para proporcionar esta información adicional a HaikuDepot, una aplicación de escritorio para administrar paquetes.
